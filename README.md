# pdb-to-fasta

A simple C library to extract the amino acid sequence from a file in PDB (Protein data bank) format and output to a FASTA format file.

## Requirements

- A compiler (GCC, Clang, etc.)
- CMake (3.6 or later)

## Getting started

Obtain the source code, either by downloading a zip/tar compilation of the source code or clone with git.
Run CMake,

```sh
$ cd /path/to/code
$ cmake -B build --config Release --target all .
```
